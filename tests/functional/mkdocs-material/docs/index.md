# Plissken Functional Test

This is a functional test site for validating plissken's documentation rendering with MkDocs Material.

## What's Being Tested

- **Theme Adapter**: CSS variables resolve correctly (`var(--md-*)`)
- **Badges**: async, visibility, source type badges render with proper styling
- **Code Blocks**: Syntax highlighting works
- **Parameter Tables**: Markdown tables render correctly
- **Dark Mode**: Theme toggle works and styles update
